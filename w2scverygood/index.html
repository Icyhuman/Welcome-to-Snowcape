Welcome to the interactive map for Snowcape, my Animal Crossing island. The site is still a work in progress, but I recommend looking at my cool new fashion section.

I played this game actively for a couple of years after it came out, and I still check back on it occationally. Over that time I've made a bunch of stuff, and taken over a thousand screenshots. I figured I should do something with all those screenshots, and ended up combining it with my programming skills to create this cool map. Please enjoy. I also have another website at icyhuman.github.io if ya want to check it out.
The mapping was done with an awful mapping tool built with hopes, dreams and duct tape. To peep the horror, check out the github repo.
